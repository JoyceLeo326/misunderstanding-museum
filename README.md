# Intent 误会博物馆

> 语言可以被翻译，关系需要被理解。

[查看线上案例](https://intent-misunderstanding-museum.vercel.app/) · [打开 4 页案例 PDF](https://intent-misunderstanding-museum.vercel.app/assets/intent-misunderstanding-museum-case-study.pdf)

## 项目概述

“Intent 误会博物馆”是一套面向小红书的跨文化内容增长案例。项目从“字面翻译已经正确，关系语境仍可能被误读”的具体瞬间出发，把 AI 实时翻译能力放进跨国情侣、海外生活、K-pop 同好和网络语境中的真实内容场景。

## 核心交付

- 1 个长期品牌内容 IP：Intent 误会博物馆
- 4 个固定内容展厅：关系、异乡生活、同好暗号、网络语境
- 12 张首批选题卡与完整 Hero 样稿
- UGC 投稿、匿名授权、审核与二次分发流程
- KOC 角色分工、内容 Brief 与合作边界
- 4 组单变量实验和从内容到首次跨语言会话的指标框架

## 线上体验

案例站包含滚动揭示、藏品自动轮换、路径脉冲、漂浮标签、横向跑马灯、卡片指针倾斜和交互式投稿演示；同时支持 prefers-reduced-motion，不会保存或上传表单输入。

## 内容结构

每件“藏品”都使用同一套编辑骨架：

原话 → 对方听成 → 后来才知道 → 共同注释

首件藏品 MIS-001 以“吃了吗？”为例，展示一句话如何被正确翻译，却因为成长经验和关系语境产生完全不同的理解。

## 技术与交付

- Semantic HTML5 / CSS / Vanilla JavaScript
- 响应式桌面与移动端布局
- 无框架、无构建步骤、无后端和外部字体请求
- Vercel 生产部署
- GitHub 单文件公开版本，便于直接审阅源码

## 项目边界

网站已公开部署；内容活动本身尚未执行，不使用真实用户投稿、内部数据或虚构投放结果。Hero 故事是用于展示内容方法的虚构复合样稿，不代表 Intent 官方合作或用户证言。
