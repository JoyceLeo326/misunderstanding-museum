# 04 · UGC and KOC playbook

**Status:** Proposed operating model · No submissions collected · No creators contacted

## Participation idea

> 请把那句没能被解释明白的话，捐给误会博物馆。

用户不是“投稿一篇文章”，而是捐赠一件藏品。这个动作比开放式“分享你的跨文化故事”更具体，也能直接获得品牌编辑需要的结构化素材。

## Submission template

```text
藏品名称：
当时的原话：
涉及语言：
我真正想表达：
对方当时怎样理解：
后来怎样解释清楚：
是否同意匿名收录：
是否拥有相关聊天与素材授权：
```

## Recommended entry points

### Comments

- 优点：参与门槛最低，能直接制造共同注释。
- 风险：不适合提交私密信息；完整故事不应要求在公开评论区写出。
- 使用方式：评论只留“原话+涉及语言”，品牌私信获得后续授权。

### Private-message keyword

- 关键词建议：`捐藏品`
- 优点：适合承接较长故事，也可以发送正式授权说明。
- 风险：私信仍不是隐私数据库，需要定义删除、保存和访问权限。

### External form

- 只在真实项目具备隐私说明、数据最小化、删除渠道和权限控制后启用。
- 不采集与内容策展无关的身份、位置或联系方式。
- 敏感对话截图设为可选，且必须确认双方授权。

公开案例网站中的表单只是前端流程演示，不上传、保存或处理任何输入。

## Curation workflow

1. **Triage** — 检查是否属于跨文化语境，而非普通关系冲突。
2. **Permission** — 确认投稿者对故事和附件拥有分享权限。
3. **Anonymization** — 移除姓名、账号、头像、地点、机构和敏感时间。
4. **Clarification** — 向投稿者核对原话、双方理解和后续结果。
5. **Stereotype review** — 删除把单一经验概括成民族性格的表述。
6. **Editorial adaptation** — 改写为统一藏品结构，保留投稿者个人口吻。
7. **Final approval** — 发布前将最终文本回传投稿者确认。
8. **Community annotation** — 邀请不同背景用户补充，不宣布唯一答案。
9. **Reuse consent** — 二次投放、达人共创或跨平台发布需单独确认范围。

## Community response examples

### When a user submits a useful phrase

> 这句话很适合进馆。我们先不判断谁理解得“对”，更想知道：当时你真正想表达什么？如愿意匿名分享，可以私信关键词“捐藏品”，我们会先发送授权与隐私说明。

### When users disagree about meaning

> 这正是“共同注释”的意义：同一句话进入不同地区、家庭和关系阶段，可能承担不同功能。欢迎说说你的使用场景，不需要替所有人下结论。

### When a comment stereotypes a culture

> 我们保留对具体经历的讨论，但不把个体行为概括成某个国籍或文化的固定性格。可以把“他们都……”改成“我这一次遇到的是……”。

## Creator relay

### Role map

| Creator type | Provides | Should avoid |
|---|---|---|
| Cross-cultural couple/family KOC | Relationship tension and real dialogue | Turning private conflict into spectacle |
| International student/overseas Chinese KOC | Specific daily-life context | Generalizing a country or culture |
| K-pop/multilingual fandom creator | Slang, symbols and community language | Producing a dry language lesson |
| Language/culture creator | Credible context explanation | Declaring one universal correct meaning |
| AI/product creator | Product recognition and use case | Taking over the emotional content layer |

## Creator screening rubric

建议用六项评分卡，而不是只看粉丝量：

1. **Audience fit** — 粉丝国家、语言、生活阶段是否与目标人群相符。
2. **Persona stability** — 创作者是否长期在该语境中表达，而非临时蹭题。
3. **Comment quality** — 评论区是否有真实讨论、补充和故事，而非单一表情。
4. **Commercial naturalness** — 过往合作内容能否保留个人表达。
5. **Delivery reliability** — 沟通、脚本、审稿与发布时间是否稳定。
6. **Safety risk** — 是否存在隐私、歧视、攻击性或未经授权素材风险。

正式执行前，应由团队确定权重、最低分与一票否决项。

## One-page KOC brief

### Campaign

Intent误会博物馆 · `#翻译不出的话`

### Task

> 从一句你真正经历过的跨文化误会出发，把“为什么会误会”和“后来怎样继续聊”讲完整。

### Must include

1. 一句真实原话。
2. 双方当时不同的理解。
3. 后来怎样解释清楚。
4. 一个邀请粉丝分享自身经历的问题。

### Product appearance

Intent只在“双方继续用各自语言聊天”的自然节点出现。不得宣称它识别情绪、读懂潜台词或保证消除误会。

### Suggested deliverables

- 1篇6—8页小红书图文或1条45—60秒短视频；
- 2版封面；
- 1条置顶互动问题；
- 合作关系与商业利益披露；
- 明确的品牌二次编辑和分发授权范围。

### Do not

- 不编造跨国恋情、留学或海外生活经历。
- 不未经同意展示聊天截图、照片或声音。
- 不使用统一广告口吻硬夸产品。
- 不虚构使用结果或转化数据。
- 不用国籍和文化背景解释一个人的全部性格。

## Content loop

```text
品牌Hero首展
  → 种子KOC捐出第一批真实藏品
  → 评论区共同注释
  → 私信关键词/表单投稿
  → 授权、匿名化和策展
  → 品牌号发布“本周新入馆”
  → 原作者与同类KOC二次传播
  → 自然表现较优内容进入小额测试
  → 高频问题回到下一轮选题库
```

这里的“循环”只是拟议工作流，不代表已经产生UGC、合作或投放结果。
