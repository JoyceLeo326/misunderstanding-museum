## Status legend

- `Content system`：已经形成结构、模板与可直接执行的内容资产。
- `Editorial sample`：完成封面、正文、卡片与互动入口的完整内容脚本。
- `Ready to run`：流程、责任与检查项已经定义，可直接进入协作。
- `Ready to measure`：变量、指标与回填口径已经定义，可承接实际发布结果。
