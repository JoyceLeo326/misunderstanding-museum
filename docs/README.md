## Status legend

- `Concept`：独立构思，未被委托。
- `Mockup`：为展示执行能力制作的内容化样稿。
- `Proposed`：建议流程，尚未实际运行。
- `Not measured`：没有真实数据，不填写估计值。

## Public scope
